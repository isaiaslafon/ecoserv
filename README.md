# Ecoserv

La idea es tener un repo que tenga por carpeta cada servicio orientado a dicha VM.

Y tener centralizado una isntancia de Nginx para reverse proxy y un PostgreSQL para metadata de los servicios.

## Proxy Reverso:

 - Nginx

## Monitoreo:

 - Kuma Uptime
 - Netdata

## Versionado:
